# Audio Mixer

브라우저에서 동작하는 개인 음향 편집 / 믹싱 도구입니다. Web Audio API 기반.

## 기능
- **7-band EQ** — 주파수별 볼륨 조절 (SUB ~ TREBLE)
- **이펙트** — 리버브, 딜레이, 컴프레서, 디스토션
- **FILE 모드** — 오디오 파일을 불러와 편집
- **LIVE 모드** — 탭/시스템 오디오를 실시간 캡처해 같은 이펙트 체인에 통과
  - 원본 소리는 자동 차단(`suppressLocalAudioPlayback`)되어 처리음만 깔끔하게 재생
  - 네온 오실로스코프로 파형 시각화
- **다크 / 라이트 테마** 전환

## 사용
GitHub Pages 주소로 접속하면 바로 실행됩니다. (HTTPS이므로 LIVE 캡처도 동작)

> LIVE 캡처는 Chrome / Edge 권장. 캡처는 "접속한 기기 자신"의 오디오를 대상으로 합니다.
