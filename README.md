# Audio Mixer

브라우저에서 동작하는 개인 음향 편집 / 믹싱 도구입니다. Web Audio API 기반.

## 기능
- **7-band EQ** — 주파수별 볼륨 조절 (SUB ~ TREBLE)
- **이펙트** — 리버브, 딜레이, 컴프레서, 디스토션
- **FILE 모드** — 오디오 파일을 불러와 편집
- **LIVE 모드** — 탭/시스템 오디오를 실시간 캡처해 같은 이펙트 체인에 통과
  - 원본 소리는 자동 차단(`suppressLocalAudioPlayback`)되어 처리음만 깔끔하게 재생
  - 네온 오실로스코프로 파형 시각화
- **다크 / 라이트 테마** 전환

## 사용
GitHub Pages 주소로 접속하면 바로 실행됩니다. (HTTPS이므로 LIVE 캡처도 동작)

> LIVE 캡처는 Chrome / Edge 권장. 캡처는 "접속한 기기 자신"의 오디오를 대상으로 합니다.

# 미스틱 타로 (Mystic Tarot)

`tarot.html` — 라이더 · 웨이트 메이저 아르카나 22장으로 보는 온라인 타로점.

## 기능
- **실제 카드 디자인** — 라이더 · 웨이트 · 스미스(1909, 퍼블릭 도메인) 카드 이미지를 Wikimedia Commons에서 로드
- **3가지 스프레드** — 원 카드 / 쓰리 카드(과거·현재·미래) / 상황·행동·결과
- **셔플 & 픽** — 카드를 섞고 직접 뽑는 인터랙션, 뒤집기(flip) 애니메이션
- **정방향 / 역방향** — 무작위 역방향, 방향별 키워드 + 해석 제공
- **종합 메시지** — 뽑힌 카드 조합에 따른 전체 흐름 요약
- 단일 HTML 파일, 빌드 불필요. 이미지 로드 실패 시 카드별 대체 디자인 표시

## 사용
GitHub Pages 주소 뒤에 `/tarot.html` 을 붙여 접속합니다.
